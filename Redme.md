
PI – Presencial – Página de cadastro de cliente ou usuário 
A validação de um formulário é muito importante para que o usuário evite de fazer um preenchimento de cadastro de forma incorreta!  

Crie uma página que terá um formulário de cadastro, que possa ser um cliente do seu site ou um usuário do sistema! 


Onde será necessário fazer a validação de cada item do formulário.  

O que terá no formulário:  

Nome  

CPF ou CNPJ  

Telefone  

Celular  

Endereço(Separando em cada caixa de texto):  

CEP  

Logradouro(Rua, avenida e etc)  

Numero  

Complemento  

Bairro  

Cidade  

Estado  

Botões de cadastrar  

  

Validação de formulário  

Pode utilizar Jquery na Validação ou somente Javascript com Expressão Regular ou Regex. Dependendo do campo é possível utilizar o Required do HTML, para fazer a obrigatoriedade do preenchimento do campo e Placeholder para mostrar na caixa de texto como preencher os campos.  

Validar preenchimento do campo nome  

Validar preenchimento do CPF ou CNPJ, com valores válidos!  

Validar preenchimento do CEP, com valores válidos.  

Utilizar no Cep a API com uso de AJAX para trazer o resultado dos campos. (Exemplo nas páginas 101 à 104).  

Os campos de Logradouro, Bairro, Cidade, Estado, devem estar desabilitados pois eles devem ser preenchidos de forma automática. Deixando somente número e complemento habilitados para preenchimento e o numero dever ser preenchimento obrigatório.  

Botão cadastrar deve funcionar somente se os campos forem preenchidos de forma correta.  

Utilizar um JSON para capturar os dados do formulário, que pode ser exibido na mesma página ou em uma página diferente! Listar a informação capturada!  

  

  

Resultado da Busca do Cep  

Imagem Digitando o Cep  

 



Imagem aparecendo o endereço automaticamente depois de preencher o Cep  

  

  

  

Exemplo código Busca cep:  https://viacep.com.br/exemplo/javascript/  

  

  

Critérios de Avaliação  

Design da página desenvolvida(HTML e CSS)  

Organização das pastas e arquivos(HTML,CSS, imagens e etc) mostrado na UC7.  

Validações do formulário devem estar funcionando de forma correta.  

O funcionamento da Busca de CEP em AJAX   

Informações capturadas via JSON, sendo exibidas de forma correta

Aplicar os princípios de Gestão de Qualidade como: foco no cliente, envolvimento de pessoas , abordagem de processos e abordagem sistêmica

Conduzir as emoções dos usuários em relação ao sistema
